Variáveis - Lógica de programação com JavaScript

Declaração de Variáveis

O nome das variáveis possuem regras para sua declaração
JavaScript é uma liguagem case sensitive
Não pode conter caracteres
Não pode iniciar com números
Não pode ter espaços
Pode iniciar com o caractere _ underline
Não pode ser palavras reservadas da linguagem
Dica: Usar os nomes da variáveis em português!!!

Palavras Reservadas

São as palavras chaves identificadores pré reservados, como por exemplos os comandos da linguagem de programação.
Exemplo de palavras reservadas: abstrat, byte, int, import, boolean, default, char, do, for, goto, case, continue, delete, etc...

Tipos de dados JS - Data types

null - Não valor, ou inexistente de valor associado a uma variável
undefined - Conteúdo de variável não iniciada
boolean - Verdadeiro (true) ou falso (false)
string - Para sequência de zeros e caracteres, normalmente representa textos
number - Números inteiros e flutuantes
NaN - Not a Number, número inválido
symbol - Obsoleto... usado para objetos
object - Para coleção de propriedades
